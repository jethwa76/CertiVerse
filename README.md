# 📜 CertiVerse

> **A modern, responsive certificate showcase website built using HTML, CSS, and JavaScript.**

CertiVault is a personal portfolio website designed to organize and display engineering certificates, achievements, internships, workshops, hackathons, and technical skills in a clean and professional manner. It serves as a digital achievement portfolio that can be shared with recruiters, professors, and peers.

---

## ✨ Features

* 🎨 Modern and responsive UI
* 📱 Mobile-first design
* 🌙 Dark theme
* 🖼️ Certificate gallery with preview modal
* 🔍 Category-based certificate filtering
* 📜 Achievement timeline
* 💻 Skills showcase
* 📬 Contact section with form validation
* ✨ Smooth scrolling and animations
* ⚡ Built with pure HTML, CSS, and JavaScript (No frameworks)

---

## 🛠️ Tech Stack

* HTML5
* CSS3
* JavaScript (Vanilla)
